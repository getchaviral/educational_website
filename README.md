# CuriousCubsClub - Educational Website

## Overview

CuriousCubsClub is an educational platform designed to support learning for Kashmiri migrant communities, including Gujjars and Bakarwals. The website provides subject-wise educational content, class-based learning resources, and interactive quizzes through a simple and accessible interface.

## Mission

Our mission is to empower underprivileged and displaced students through education. We believe every child deserves access to quality learning opportunities regardless of their background or circumstances.

By promoting education among migrant communities, we aim to help students build knowledge, confidence, and a brighter future.

## Features

* Interactive educational platform
* Class-wise learning materials
* Mathematics modules

  * Addition
  * Subtraction
  * Multiplication
  * Shapes
* English learning section
* Hindi learning section
* Interactive quizzes
* User registration and login pages
* Responsive design

## Installation

1. Clone the repository

```bash
git clone https://github.com/getchaviral/educational_website.git
```

2. Navigate to the project folder

```bash
cd educational_website
```

3. Open `index.html` in your browser.
